HTML 简介

HTML 是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

HTML 实例

<!DOCTYPE html>
<html>
<head>
    <title>页面标题</title>
</head>
<body>
    <h1>我的第一个标题</h1>
    <p>我的第一个段落。</p>
</body>
</html>

注意： 对于中文网页需要使用 <meta charset="utf-8"> 声明编码，否则会出现乱码。

HTML 基础

HTML 文档由嵌套的HTML元素构成。

HTML 文档结构

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>文档标题</title>
</head>
<body>
    <h1>这是一个标题</h1>
    <p>这是一个段落。</p>
</body>
</html>

HTML 基本概念

  • <!DOCTYPE html> - 声明文档类型
  • <html> - HTML文档的根元素
  • <head> - 包含文档的元信息
  • <body> - 包含可见的页面内容

HTML 元素

HTML 元素指的是从开始标签到结束标签的所有代码。

HTML 元素语法

  • HTML 元素以开始标签起始
  • HTML 元素以结束标签终止
  • 元素的内容是开始标签与结束标签之间的内容
  • 某些 HTML 元素具有空内容
  • 空元素在开始标签中进行关闭
  • 大多数 HTML 元素可拥有属性

HTML 元素示例

<p>这是一个段落</p>
<a href="https://example.com">这是一个链接</a>
<img src="image.jpg" alt="图片描述">

CSS 简介

CSS (层叠样式表) 是一种用来表现HTML或XML等文件样式的计算机语言。

为什么使用 CSS?

  • CSS 节省了大量工作，可以同时控制多个网页的布局
  • 外部样式表存储在 CSS 文件中，提高了代码的可维护性
  • 样式与内容分离，使HTML更专注于结构
  • 提供了丰富的样式控制能力

CSS 实例

body {
    background-color: lightblue;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: verdana;
    font-size: 20px;
}

CSS 选择器

CSS 选择器用于"查找"（或选取）要设置样式的 HTML 元素。

CSS 选择器类型

选择器 示例 描述
元素选择器 p { color: red; } 选择所有<p>元素
类选择器 .intro { font-weight: bold; } 选择class="intro"的元素
ID选择器 #header { background: #333; } 选择id="header"的元素
